QUICK START COMMANDS
================================================================================

1. Install dependencies:
   pip install -r requirements.txt

2. Setup environment (optional):
   cp .env.example .env
   # Edit .env to add API keys if desired

3. Ingest and index documents:
   python scripts/01_ingest_documents.py
   python scripts/02_build_indexes.py

4. Run the Streamlit app:
   streamlit run app.py

5. Run evaluation (optional):
   python scripts/05_run_evaluation.py

6. Train LoRA (optional):
   python scripts/03_generate_finetune_data.py
   python scripts/04_train_lora.py

7. Run tests:
   pytest tests/

KEY DESIGN DECISIONS
================================================================================

1. Architecture
   - Modular design with clear separation of concerns
   - LangGraph for multi-agent coordination
   - Hybrid RAG for both course and external knowledge

2. Data Management
   - Pydantic models for type safety
   - JSONL format for large datasets
   - ChromaDB for vector storage

3. Safety & Ethics
   - Safety agent checks every query
   - Source validation against approved domains
   - Citation checking for grounding
   - Refusal patterns for harmful requests

4. Flexibility
   - Deterministic local LLM fallback for testing without API keys
   - Environment-based configuration
   - Pluggable retrieval strategies
   - Optional online search

5. Evaluation
   - Baseline comparisons
   - Comprehensive metrics

   - Safety testing
